## Button

- Severities: `primary`, `secondary`, `success`, `info`, `warn`, `danger`, `help`, `contrast`
- Variants: `outlined`, `text`, `link`, `raised`, `rounded`
- Size: `small`, `large`
- Icons: `[icon]`, `[iconPos]`
- Loading: `[loading]`, `[loadingIcon]`
- Templating the label vs using `[label]`
- Disabled vs loading semantics

## SpeedDial

- The floating-action-button pattern
- Direction modes
- Type: `linear` / `circle` / `semi-circle` / `quarter-circle`
- Mask backdrop option

## SplitButton

- Primary action + dropdown of secondary actions
- `model` array shape
- Severity propagation to the dropdown items

## Accessibility

- All three include keyboard handlers and proper ARIA out of the box
- `aria-label` required when icon-only — call this out explicitly
